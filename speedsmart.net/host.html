Host a Server

Our goal for SpeedSmart is to have a reliable high quality speed test server in as many cities around the globe as we can. Adding your server to our network will help us achieve this goal.

Each host sponsoring a speed test server is provided with a login to our statistics and reporting system. When logged in you can view all speed tests run againt your server and get valuable global statisits for the past 24 hours, 7 days and 30 day periods. This is great for ISPs or data centers that would like a more real world perspective of how there network performs.

To get started download the file here, unzip it then upload it to your server. It contains one 100 MB file and a small PHP file. Once uploaded fill out the form, we will contact you letting you know your server was added.

To say thanks for sponsoring a server we will offer a link to your website at the bottom of this page as well as on both our Twitter and Facebook.
